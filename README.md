# Erdem - Content Creator Landing Page

A stunning, animated landing page for Erdem, a social media content creator seeking brand collaborations.

## Features

- Modern, visually impressive design with smooth animations
- Hero section with gradient animations and social links
- Social metrics display with counter animations
- Content showcase with masonry grid layout
- Brand carousel with auto-scroll
- Contact CTA section
- Fully mobile responsive
- Optimized for performance

## Tech Stack

- Next.js 14 with App Router
- TypeScript
- Tailwind CSS
- Framer Motion for animations
- Lucide React for icons

## Getting Started

### Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view in browser.

### Build

```bash
npm run build
npm start
```

## Deployment on Vercel

This site is ready to deploy on Vercel:

1. Go to [vercel.com](https://vercel.com)
2. Import the GitHub repository: `mrgede-landing`
3. Configure custom domain: `mrgede.com`
4. Deploy!

The site will be live at:
- Preview: `https://mrgede-landing.vercel.app`
- Production: `https://mrgede.com`
